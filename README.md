# Kiddy - IT-школа для детей

Веб-сайт для детской IT-школы Kiddy в Минске.

## Описание

Сайт создан для представления программ обучения детей программированию, созданию игр, 3D-моделированию и работе с AI.

## Структура проекта

- `main.html` - главная страница сайта
- `css/modal.css` - стили для модальных окон
- `js/modal.js` - JavaScript для модальных окон и интерактивных элементов
- `main_files/` - ресурсы главной страницы (изображения, стили, скрипты)
- `1modal_files/`, `2modal_files/`, `3modal_files/`, `4modal_files/` - ресурсы для модальных окон уровней

## Запуск

Просто откройте `main.html` в браузере. Все ресурсы локальные, интернет не требуется.

## Функциональность

- ✅ Модальные окна для каждого уровня обучения
- ✅ FAQ аккордеон
- ✅ Плавная прокрутка
- ✅ Адаптивный дизайн
- ✅ Анимации и переходы
- ✅ Форма подписки на новости
