# Рекомендации по картинкам для карточек уровней

## Общие требования:
- **Стиль**: Современный, яркий, дружелюбный
- **Цветовая схема**: Cherry Red (#9B1B30) + кремовый/белый фон
- **Настроение**: Вдохновляющее, позитивное, детское (но не инфантильное)
- **Размер**: 1200x800px (16:10) или 1200x600px (2:1)
- **Формат**: JPG или PNG, оптимизированные (< 200KB)

## Уровень 1: "Я управляю компьютером"
**Промпт для нейронки:**
```
A happy child (8-10 years old) sitting at a colorful computer, creating a simple game in Scratch. Bright, modern workspace with cherry red accents. Friendly, inspiring atmosphere. Digital art style, vibrant colors, clean background.
```

**Альтернативные варианты:**
- Ребёнок с ноутбуком, на экране визуальные блоки Scratch
- Дети работают в команде за компьютерами, создают 3D-модели
- Яркая студия с детьми, которые создают свои первые проекты

## Уровень 2: "Я делаю игры"
**Промпт для нейронки:**
```
A creative child (10-12 years old) developing a game on a computer screen showing Construct 3 interface. Python code visible on another screen. Modern gaming setup with cherry red and cream colors. Energetic, creative atmosphere.
```

**Альтернативные варианты:**
- Ребёнок за компьютером, на экране игровой движок
- Дети тестируют свою игру на планшете/телефоне
- Игровая студия с детьми, создающими персонажей и уровни

## Уровень 3: "Я почти как разработчик"
**Промпт для нейронки:**
```
A focused teenager (13-15 years old) coding in Python, AI chatbot interface visible, 3D game engine on screen. Professional but friendly workspace. Cherry red accents, modern tech setup. Confident, skilled atmosphere.
```

**Альтернативные варианты:**
- Подросток работает с AI-инструментами (ChatGPT, DALL-E)
- Создание 3D-игры в Godot Engine
- Разработка бота или автоматизации

## Уровень 4: "Я готов к профессии"
**Промпт для нейронки:**
```
A confident young developer (15-17 years old) working on professional project, GitHub visible, VS Code with React Native code. Modern office setup with cherry red branding. Professional, inspiring, ready-for-career atmosphere.
```

**Альтернативные варианты:**
- Молодой разработчик презентует свой проект
- Работа над мобильным приложением
- Портфолио проектов на экране

## Цветовая палитра для изображений:
- **Основной**: Cherry Red #9B1B30
- **Светлый**: #C23B4F (cherry-light)
- **Фон**: Кремовый #F2F0EB (alabaster) или белый
- **Акценты**: Золотой/янтарный для деталей

## Стиль изображений:
- **Не использовать**: Реалистичные фото детей (проблемы с правами)
- **Использовать**: Иллюстрации, 3D-рендеры, стилизованные изображения
- **Настроение**: Позитивное, вдохновляющее, современное
- **Композиция**: Фокус на действии, технологии, творчестве

## Инструменты для генерации:
- **Midjourney**: Лучше всего для стилизованных иллюстраций
- **DALL-E 3**: Хорошо для детальных промптов
- **Stable Diffusion**: Для полного контроля над стилем
- **Leonardo.ai**: Бесплатный вариант с хорошим качеством
